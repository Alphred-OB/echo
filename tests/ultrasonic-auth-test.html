🔊 Ultrasonic Auth — Hardware Test

Step 1 of the prototype: can your laptop ↔ phone pass a nonce over near-ultrasound?
—
Nonce will appear here, then play as sound.
—
Tap start, allow microphone, hold phone ~30 cm – 2 m from the laptop.
How to test:
0. One-time setup: download ggwave.js and save it in the same folder as this page.
1. Serve this file (mic needs a secure context): on the laptop run python3 -m http.server 8000 and open http://localhost:8000/ultrasonic-auth-test.html.
2. For the phone you need HTTPS — easiest: push this file to GitHub Pages, or run npx ngrok http 8000 and open the https URL on the phone.
3. Phone: tab "Listen" → start. Laptop: tab "Send" → transmit.
4. If the nonce matches on both screens — your hardware can do ultrasound. 🎉
5. Record what works: max distance, volume needed, behavior with music/talking in the room.

Quick same-laptop test: open two tabs on localhost, one sending, one listening (works, since speaker and mic are on the same machine).
